## Project Overview
This project implements a relational database for a hotel management use case.  
It manages room types, rooms, guests, and bookings, and supports data integrity, CRUD operations, and reporting queries.

## Features
- Normalized relational schema
- Primary/foreign keys and constraints
- Business validation checks (booking dates, positive prices)
- Indexed columns for query performance
